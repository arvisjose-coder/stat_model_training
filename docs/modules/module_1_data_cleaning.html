1 Vue d’ensemble du module

Le Module 1 sert à apprendre à connaître les données avant de choisir un modèle statistique.

L’idée centrale est simple :

On ne commence pas par choisir un modèle.
On commence par comprendre les données.

2 Objectifs à maîtriser

À la fin de ce module, je dois être capable de :

  • importer un jeu de données dans R ;
  • inspecter sa structure ;
  • reconnaître les types de variables ;
  • nettoyer les noms de colonnes ;
  • détecter les doublons ;
  • repérer les valeurs manquantes ;
  • transformer les variables au bon format ;
  • réaliser une analyse exploratoire ;
  • préparer les données pour les modules suivants.

11 Pourcentage de valeurs manquantes

Code 
missing_summary <- df_traits %>%
  summarise(across(everything(), ~ sum(is.na(.)))) %>%
  pivot_longer(
    cols = everything(),
    names_to = "variable",
    values_to = "n_missing"
  ) %>%
  mutate(
    pct_missing = 100 * n_missing / nrow(df_traits)
  ) %>%
  arrange(desc(pct_missing))
